# [Wordle](https://www.powerlanguage.co.uk/wordle/) Solver

### Usage

Run the `main` method in `Scwordle.scala` and interact at the console.

Play the words given.

After each play give the response in the following format:

* v: correct (think ✔)
* x: nowhere (think ❌)
* ~: somewhere else
   
So for example

⬛🟨⬛🟩🟨

would be

```
x~xv~
```
